# Swirlfeed
### 소셜미디어 사이트(Social media Website)

#### :computer: URL: http://swrilfeed.dothome.co.kr/  <br>
#### :pencil2: 제작과정(production process): https://blog.naver.com/slgktn92/221410360108

###### Front
* HTML/CSS/JS
* Jquery
* Bootstrap
* Bootbox

###### Server
* PHP
* 닷홈(dothome) - Web hosting
* MySQL
* AJAX

###### Function
* 회원가입 / 로그인 (Sign up / Sign in)
* 글쓰기(Youtube Video, Image 업로드 가능) / 글리스트(좋아요) / 삭제 (Create, Read(&Like), Delete)
* 회원정보 수정 / 메세지 / 친구 추가 / 알림 기능 (Update user information / Message / Add friend / Notification)
* 코멘트(Comment with iframe) / 트렌드 단어 보여주기(Show trends word) / 유저 검색 (User search)
* 무한 스크롤링(Infinite scrolling)
